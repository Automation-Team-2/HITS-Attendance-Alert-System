# HITS Attendance Alert System — Architecture Document

This folder contains the official architecture document for the HITS Attendance Alert System.

## File

- **`HITS_Attendance_Architecture_Document.docx`** — A comprehensive Word document covering:
  1. **System Architecture** — High-level component design and interaction flow
  2. **Database** — 12-table MySQL schema, ER diagram, primary/foreign keys, and attendance formulas
  3. **Frontend** — Django web framework, JavaScript AJAX logic, and NodeJS build tooling
  4. **Backend** — FastAPI REST API, endpoint definitions, and asynchronous MySQL access
  5. **Automation** — Daily APScheduler tasks, WhatsApp/Email alert pipeline, and spam prevention
